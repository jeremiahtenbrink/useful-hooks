del

del: function
  • Defined in hooks/useAxios.ts:177

Function

Call this function to make a delete http request using axios. The response from the axios call will be placed in index 1 of the returned array from useAxios call.

param

string, either the remaining portion of the url or the url of http request.

param

object to pass to the server if one is requried. If not use empty object.

param

boolean, determines if you want to use the auth axios call or not. setAxiosAuthConfig must have already been called in your app.
