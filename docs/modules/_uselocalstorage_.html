    Custom Hook

    This hook makes setting values to local storage easy.

    Parameters

    • key: string

      Local storage key to be used for this item to be stored in local storage.

    • Optional initialValue: any

      Initial value if there isn't a value in localstorage.

    Returns [LocalStorageValue, SetValue, RemoveValue]

    • Returns a array. Index 0 is the value of the localStorage. Index 1 is a function used to change the value of the item in local storage. Index 2 is a function used to remove the item from local storage.
