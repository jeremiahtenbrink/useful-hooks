  • useForm(onSubmit: function, formDefaultValues?: __type): [FormValues, HandleFunctions]
    • Defined in useForm.ts:19

    Custom Hook

    Call useForm to handle all state changes for any form. Returns an array. Index 0 is the values from the form. Index 1 is a object containing all the functions to interact with the form inputs.

    Parameters

    • onSubmit: function

      callback function to be called once the form is submited.

        • (IValues: any): void

        • Parameters

          • IValues: any

          Returns void

    • Default value formDefaultValues: __type = {}

      object containing the default values of the form.

    Returns [FormValues, HandleFunctions]

    • Form values is a object with key value pairs. Keys being the input names and values being the value of the users input. The HandleFunctions object is a object containing the functions change, submit, and clear.
