FormValidate

FormValidate: function
  • Defined in useForm.ts:96

Callback Function

If this function is not undefined then it will be called on every onChange function call. This callback function will be called with the name of the input and the value of the input. The user can decide to return either a empty string ( No Error ) or a string with the error message.

param

name - The name of the input field.

param

value - The value of the input field.

returns

Must return either a blank string indicating no error or a error message as a string value.

      Returns string

    • Defined in useForm.ts:23

    Custom Hook

    Call useForm to handle all state changes for any form. Returns an array. Index 0 is the values from the form. Index 1 is a object containing all the functions to interact with the form inputs.

    Parameters

    • onSubmit: OnSubmit

      onSubmit - Callback function to be called once the form is submitted.

    • Optional formDefaultValues: FromDefaultValues

      formDefaultValues - Object containing the default values of the form.

    • Optional formValidate: FormValidate

      formValidate - Callback function for validating the forms values.

    Returns [FormValues, HandleFunctions]

    • Form values is a object with key value pairs. Keys being the input names and values being the value of the users input. The HandleFunctions object is a object containing the functions change, submit, and clear.
